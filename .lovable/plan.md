

# Генерация логотипа Flamelix в стиле Claude

## Задача
Сгенерировать текстовый логотип "FLAMELIX" в стиле Claude.ai — минималистичный, с антиквенным шрифтом (serif), теплой палитрой.

## Что будет сделано

### 1. Генерация логотипа через AI (Gemini)
Создание edge function, которая вызовет Lovable AI (google/gemini-2.5-flash-image) с промптом для генерации логотипа:
- Текст "FLAMELIX" в минималистичном serif-стиле
- Теплая палитра: терракотовый (#C4704B) на прозрачном/белом фоне
- Стиль вдохновлен Claude.ai — чистый, элегантный, без лишних элементов

### 2. Сохранение и замена
- Сгенерированное изображение будет сохранено как новый файл `src/assets/flamelix-logo-new.png`
- Обновление импортов в Header и Footer для использования нового логотипа
- Возможно потребуется убрать CSS-фильтр `invert`, если логотип будет цветным

## Ограничения
- AI-генерация текстовых логотипов не всегда идеально передает буквы — может потребоваться несколько итераций
- Для профессионального логотипа лучше использовать векторный редактор (Figma, Illustrator), но для прототипа AI-генерация подойдет

## Технические детали

### Изменения в файлах
- `src/components/Header.tsx` — обновить путь к логотипу, убрать/скорректировать `invert` фильтр
- `src/components/Footer.tsx` — обновить путь к логотипу
- Новый файл с логотипом в `src/assets/`
